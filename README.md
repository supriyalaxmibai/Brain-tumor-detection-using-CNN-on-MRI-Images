# Brain Tumor Detection using CNN and MobileNetV2 on MRI Images

## Overview
This project presents an automated brain tumor detection system using Convolutional Neural Networks (CNN) and the MobileNetV2 architecture on MRI brain images. The proposed system classifies MRI scans into tumor and non-tumor categories using deep learning techniques.

## Features
- MRI image preprocessing
- Data augmentation
- MobileNetV2 transfer learning
- Tumor and non-tumor classification
- Accuracy and loss visualization
- Confusion matrix evaluation

## Technologies Used
- Python
- TensorFlow / Keras
- MobileNetV2
- Matplotlib
- Seaborn
- Deep Learning

## Datasets Used
- Brain Tumor Dataset
- Brain Tumor Classification MRI
- Brain MRI Images for Brain Tumor Detection

## Project Files
- Python implementation
- Research paper
- Presentation
- Output graphs and screenshots

## Results
The proposed MobileNetV2-based model achieved effective classification performance with reduced computational complexity and reliable tumor detection capability.
